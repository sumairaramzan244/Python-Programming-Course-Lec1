## Lecture 2 — Variables in Python

In Lecture 2, I learned how to use Variables in Python.

* A variable is a container that stores data values.
* A variable stores data and we can access it anytime using its name
* We can store numbers, text, and decimals in variables and use them later in our code.
* We can print variables using comma ( , ) or plus ( + ) operator
* When using ( + ), numbers must be converted to string using str()
* If we assign a new value to the same variable, the old value is replaced
* Casting means changing data type — int(), str(), float()
* type() function tells us the data type of any variable
* Python is case-sensitive — pakistan and Pakistan are two different variables
